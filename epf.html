EPF

Projet étudiant
Développement web

EPF pour Electronic Palace Festival. Comme son nom l'indique, il s'agit d'un festival de musique electronique. Un festival fictif dans le but de développer son site web en utilisant HTML, CSS et Vuejs.

Client

MMI MontbEliard

Année

2022

Outils

Guide de style

EPF a été pensé, imaginer, construit et développer de A à Z. La première étape, trouver un nom, un lieu, et les styles ou artistes qui seront programmés.
Avec un cadre défini et un fil conducteur, il est maintenant possible d'élaborer son identité visuelle et de définir son univers graphique. Pour un festival de musique électronique, j'ai créé une identité visuelle simple et moderne.
Un site web en mode sombre composé d'éléments textuels blancs et un rouge vif pour attirer l'utilisateur aux endroits désirés. La police d'écriture, quant à elle, reste simple, deux sans-sérif avec des graisses contrastées. Une pour le titrage, l'autre pour le corps de texte.

Maquette

La maquette est une étape importante pour le bon développement du site web. Celle-ci donne un aperçu concret, elle permet également de garder une cohérence globale et de suivre le fil conducteur pour le visuel. À l'aide figma et des templates Tailwind, j'ai réalisé une maquette de 5 pages principales et 2 pages secondaires qui correspondent aux fiches artistes et aux fiches des scènes. Le template figma intègre les différents formats d'écrans pour le coté responsive, mais aussi les style de textes, de couleurs et d'espacement pour un développement futur sans contretemps grâce au framework Tailwind CSS.

Développement

Le développement web c'est déroulé sur une semaine dans le cadre d'un examen de première année. Les contraintes sont simples, utilisé HTML, CSS avec le framework Tailwind CSS et le point important, Vuejs couplé à une base de donnée. Une base de donnée qui dans mon cas m'a permis de créer, modifier ou supprimer des fiches artistes prévus dans la proggramation. Pour accéder à ses fonctionnalités il fallait avoir l'accès via une connexion, elle aussi présente dans la base de donnée.
Un exercice complexe, mais enrichissant d'un point de vue développement web.
